# claude-code

Spawn Claude Code coding sessions from a Kin and watch progress live in
the conversation.

This plugin is scaffolded. The runtime behavior (tool, card actions,
streaming) lands in a follow-up commit. Full documentation, setup steps,
and security notes will be added once the implementation is complete.

## Prerequisites (preview)

- The `claude` CLI installed system-wide: `npm install -g @anthropic-ai/claude-code`
- Either an active Claude Max subscription (run `claude login` once) or
  an Anthropic API key
